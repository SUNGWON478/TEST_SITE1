서비스 소개

기니피그 판독기는 Google Teachable Machine 기반의 이미지 분류 모델을 활용합니다. 업로드 이미지는 브라우저 내에서 분석되며, 서버 저장을 기본으로 수행하지 않습니다.

사용 방법

  1. 이미지 업로드 버튼으로 사진을 선택합니다.
  2. 모델 준비 완료 후 `판독 시작`을 누릅니다.
  3. 최종 예측 클래스와 각 클래스 확률을 확인합니다.

주의사항 및 한계

  • 모델 정확도는 학습 데이터 품질에 따라 달라질 수 있습니다.
  • 어두운 사진, 저해상도 사진은 오분류 가능성이 높습니다.
  • 결과는 엔터테인먼트/실험 목적 참고용입니다.

자주 묻는 질문

업로드한 이미지가 서버에 저장되나요?

기본 동작은 브라우저 내 처리이며, 별도 저장 API를 호출하지 않습니다.

판독 결과가 이상하게 나와요.

피사체가 중앙에 보이도록 밝고 선명한 사진으로 다시 시도해 주세요.
